Accessibility

Designed in, not added later

Keyboard and focus

Navigation, buttons and links use native controls with strong visible focus. The mobile menu closes with Escape and returns focus to its control.

Motion

Content remains visible if JavaScript fails. Reduced-motion preferences remove scrolling and entrance effects.

Content and images

Heading levels follow the document structure. Informative photographs require meaningful alternative text; decorative images use empty alternatives.
