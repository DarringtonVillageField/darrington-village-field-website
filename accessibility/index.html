Accessibility statement

Making the website easier for everyone to use

Darrington Village Field Trust wants as many people as possible to be able to use this website, regardless of technology, ability or circumstance.

Draft for trustee review before launch · Prepared 17 July 2026

Our aim

A clear and accessible website

We are building the website with the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA as our target.

The complete website has not yet undergone its final accessibility testing, so we are not claiming full conformance at this stage. We will update this statement as testing is completed and improvements are made.

Current status

The website is still under development. Accessibility checks are part of the build, with a fuller review planned before the public launch.

Using this website

What you should be able to do

The site is being designed so that visitors can access its main information in different ways.

01

Read and enlarge text

Zoom in without text becoming hidden or requiring unnecessary sideways scrolling on a small screen.

02

Use a keyboard

Move through navigation, links, buttons and interactive features without needing a mouse.

03

Use assistive technology

Navigate a logical heading structure and understand meaningful images through text alternatives.

04

Reduce movement

Use the site with reduced animation when that preference is enabled on your device.

Built into the site

Measures we are taking

Accessibility is being considered throughout the design and build rather than added only at the end.

Structure
Semantic HTML, logical headings, landmarks and a skip link help people understand and move around each page.
Keyboard access
Interactive features are intended to work by keyboard and show a visible focus indicator.
Colour and type
Text, controls and backgrounds are designed with readable sizing and sufficient contrast in mind.
Motion and media
Animations respect reduced-motion settings, while slideshows provide controls and do not depend on movement alone.
Progressive enhancement
Core content remains available if JavaScript is unavailable or an enhanced feature cannot load.

Work still to complete

Known limitations during development

The following areas still require review before the website is considered ready for public launch.

Complete-site testing

Not every planned page and feature exists yet. Automated checks, keyboard testing, zoom and reflow checks, and screen-reader spot checks will be completed across the finished site.

Photograph descriptions

Alternative text and captions for images supplied through Cloudinary still need a final content review, particularly for galleries and historical photographs.

Third-party content

Links, embedded services or documents supplied by other organisations may not meet the same standard. We will review each item before adding it to the live site.

Help and alternative formats

Tell us what you need

If you cannot access information on this website, contact the Trust. Tell us which page or information you need and the format that would be more useful to you.

We will consider reasonable ways to provide the information, such as a clearer electronic version or a plain-text response.

Help us investigate

If possible, include the page address, what went wrong, and the device, browser or assistive technology you were using.

Testing and review

How we will check and improve the site

01

Before launch

Complete the planned pages, review content and images, and test navigation and interactive components.

02

Accessibility review

Run automated checks and carry out manual keyboard, zoom, reflow, contrast and assistive-technology checks.

03

Ongoing improvement

Record identified issues, prioritise fixes and update this statement when the site or its accessibility status changes.

This statement was prepared on 17 July 2026. It will be reviewed after the pre-launch accessibility assessment and following material changes to the website.
